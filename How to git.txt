git init - creates a repo
git status - checks status of repo
git add -u - Stages modified files NOT untracked files
git add -A - Stage ALL files including previously untracked files
git commit -m "Message goes here"
git branch <name> - Creates new branch (Does not check out the branch)
git checkout <name> - Checks out branch (Switch to the branch and pull from remote if not already local)
